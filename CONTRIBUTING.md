Hello!

Welcome to our project! We accept all kinds of feedback here. Before requesting or submitting anything, please make sure your comments abide by our [Code of Conduct](CODE_OF_CONDUCT.md).

If you are interested in submitting a bug or issue fix, please list the operating system, packages, environment, and versions of the submission. Then, you can fork the repo and submit it through a [pull request](https://github.com/UBC-DSCI-310-2025W2/dsci-310-group-15/pulls). Please also open an issue so we can get notified.

 If you are interested in requesting an additional feature/script/any outputs, feel free to open up an [issue](https://github.com/UBC-DSCI-310-2025W2/dsci-310-group-15/issues).

 We try to get back to pull requests ASAP (ideally within 1-2 weeks), but please be patient if it looks like we haven't done anything yet. If you haven't seen any updates after 2 weeks, you can submit a new issue/pull request.
